Array.prototype.length will give you the number of top-level elements in an Your task is to create a function deepCount that returns the number of ALL elements within an array, including any within inner-level arrays.

For example:
deepCount([1, 2, 3]); //>>>>> 3
deepCount(["x", "y", ["z"]]); //>>>>> 4
deepCount([1, 2, [3, 4, [5]]]); //>>>>> 7

The input will always be an array. 
                       function deepCount(a){
                            let sum = a.length;
                                a.forEach(el => {
                                    if(Array.isArray(el)) {
                                        sum += deepCount(el);
                                    }
                                });
                            return sum;
                        }
